Teoría de Colas - Investigación Operativa II

Modelo de línea de espera con un servidor y con llegadas Poisson y tiempos de servicio exponenciales

Este modelo describe un sistema de colas en el que las llegadas de clientes siguen un proceso de Poisson, y el tiempo de servicio tiene una distribución exponencial. En este sistema, un solo servidor gestiona las llegadas.

Modelo de línea de espera de múltiples servidores con llegadas Poisson y tiempos de servicio exponenciales

En este modelo, hay varios servidores que atienden a los clientes que llegan siguiendo un proceso de Poisson. El tiempo de servicio es exponencial, y este modelo es más eficiente cuando hay una mayor cantidad de clientes.

Esquema de nacimiento y muerte en sistema de colas (teorema de Little)

Este esquema usa un proceso estocástico de nacimiento y muerte, y se basa en el teorema de Little, que establece una relación entre el número promedio de clientes en el sistema, el tiempo promedio que pasa un cliente en el sistema y la tasa de llegada.

Modelo de línea de espera con un servidor con llegadas Poisson y tiempos de servicio arbitrarios

En este caso, las llegadas siguen un proceso de Poisson, pero los tiempos de servicio no siguen una distribución exponencial, sino que son arbitrarios, lo que añade más complejidad al análisis del sistema.

Modelo de múltiples servidores con llegadas Poisson, tiempos de servicio arbitrarios y sin línea de espera

Este modelo describe un sistema de múltiples servidores con llegadas de clientes según un proceso de Poisson. El tiempo de servicio es arbitrario y no se usa una línea de espera, lo que significa que los clientes deben ser atendidos de inmediato o se van.

Modelos de línea de espera con fuentes finitas

Este modelo considera que la cantidad de clientes posibles en el sistema es finita. Los modelos de fuentes finitas son útiles cuando se tiene una cantidad limitada de recursos o clientes potenciales.
